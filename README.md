# Eletrodrink
Programação do projeto eletrodrink, projeto de TCC realizado em 2023 para a conclusão do meu ensino médio integrado ao curso técnico de mecatrônica, o projeto foi apresentado na 58º edição da EXCUTE (Exposição Cultural e Tecnológica da ETEC Jorge Street)

# Sobre 
É um bartender automático criado com o objetivo de diminuir filas em bares, tendo em vista que pode ser pareado com o bluetooth, tem ajuste de dosagem e tem aplicativo próprio criado por meio do MIT APP INVENTOR
